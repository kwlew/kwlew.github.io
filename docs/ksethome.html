Everything you need to use and configure the plugin

Installation

  • 1. Download the plugin from GitHub
  • 2. Place the .jar file into your server's /plugins folder
  • 3. Restart your server
  • 4. Configure the plugin if needed

Commands

  • /sethome [name] — Set a home
  • /home [name] — Teleport to a home
  • /delhome [name] — Delete a home
  • /homes — List all homes

Permissions

  • ksethome.sethome — Allows setting homes
  • ksethome.home — Allows teleporting
  • ksethome.delhome — Allows deleting homes
  • ksethome.admin — Full access

config.yml

max-homes: 3
teleport-delay: 0
cooldown: 5

Notes

  • Supports multiple homes per player
  • Designed for performance and scalability
  • Future updates will include database storage
